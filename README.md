### Getting Started

1. Clone the repo.
2. Install NPM packages.
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm run dev
   ```
4. Make desired changes.
5. Open PR.

<br>

### Prerequisites

* You need to have npm.
  ```sh
  npm install npm@latest -g
  ```
